# daily-planner
Website for keeping track of upcoming events.
Website address: https://nulula.github.io/daily-planner/

## Description
This is a little helping website to keep track of upcoming events in the daily work calendar. 

## Install
N/A

## Use
On opening, the calendar will show the current hour in red, the previous hours in grey, and the future hours in green. You can add two types of events:
- with saving: when you click save button, the event will be saved to local storage and be recalled when the website is refreshed;
- without saving: if you don't click the save button, the event will disappear when you refresh the page
